# GameBoard User Guide

このドキュメントは、現在の `GameBoard` 画面の見方と基本操作をまとめたユーザー向けガイドです。

このアプリは「ルール自動処理ゲーム」ではなく、盤面を手動で操作するサンドボックスです。  
カードの移動、公開/非公開、カウンター管理、ターン進行などをプレイヤーが操作します。

## 利用環境について

- Vercel で公開しているサイトは、カード画像をダミー表示にしたバージョンです
- 公式カード画像付きでプレイしたい場合は、リポジトリをローカルで起動して利用してください

## 1. モード

- `P2P`  
  2人で接続して盤面を共有するモードです。
- `Solo Play Beta`  
  1画面で Player 1 / Player 2 の両方を操作するモードです。

## 2. 画面の見方

画面は大きく次の要素で構成されています。

- 上部ヘッダー  
  接続状態、準備ボタン、ターン情報、フェイズ、コイン/ダイス、Undo などを表示します。
- 上側ボード  
  相手側、または Solo 時の Player 2 側の盤面です。
- 下側ボード  
  自分側、または Solo 時の Player 1 側の盤面です。
- サイド操作パネル  
  `Draw`、`Mill`、`Spawn Token`、`Reset Game`、HP/PP/EP/SEP/Combo の操作があります。

各プレイヤーの主なゾーンは次の通りです。

- `Hand`
- `Field`
- `EX Area`
- `Main Deck`
- `Evolve Deck`
- `Cemetery`
- `Banish`

## 3. ゲーム開始前の流れ

準備中は上部ヘッダーに開始前用のボタンが表示されます。

基本的な流れは次の通りです。

1. 必要なら `Import Deck (.json)` でデッキを読み込む
2. `Decide Turn Order (Random)` / `Me 1st` / `Opp 1st` で先攻後攻を決める
3. `Draw Hand (4)` を押して初手4枚を引く
4. 必要なら `Mulligan` を行う
5. `Ready` を押す
6. 両者が `Ready` になったら `START GAME`

Solo モードでは `Player 1` と `Player 2` の両方に対して同じ準備を行います。

## 4. 準備中の操作

### デッキ読み込み

- `Import Deck (.json)` で JSON デッキを読み込めます
- 読み込みは準備開始前のみ可能です
- 初手を引いた後や Ready 後、対戦中は読み込めません

### 初手ドロー

- `Draw Hand (4)` で4枚引きます
- P2P では自分の手札だけが見えます
- 相手の手札は裏向きで枚数だけ確認できます

### マリガン

- 現在の仕様では `4枚全部交換` です
- `Mulligan` を押すと、戻す順番を選ぶモーダルが開きます
- 選んだ順番で山札の下に戻されます

### Ready / Start Game

- 手札確認とマリガンが終わったら `Ready`
- 両者 Ready 後に `START GAME`

## 5. 対戦中の基本操作

### カードのドラッグ

カードはドラッグしてゾーン間を移動できます。

- `Hand -> Field`
- `Field -> Cemetery`
- `Field -> Banish`
- `Field -> Deck`
- `Deck/公開ゾーン -> Field/Hand/EX`

補足:

- メインデッキのカードはエボルヴデッキへ混入しません
- エボルヴカードはメインデッキへ混入しません
- 不正な deck 混入ドラッグは、現在は「その場で動かない」動作です

### カード上へのドロップ

- 場のカードの上にカードを落とすと重ねられます
- エボルヴカードなどを重ねて使う盤面表現に使えます

### クイック操作

カードにマウスを重ねるとクイック操作が出ます。

- `REST / STAND`
- `+A / -A`
- `+H / -H`
- `+C / -C`
- `↓Bot`
- `Cemetery`
- `Banish`
- `To Evolve Deck`
- `Play to Field`

補足:

- `REST / STAND` は右クリックでも切り替えられます
- `+C / -C` は場と EX エリアのカードだけで使えます
- `To Evolve Deck` は主にエボルヴカード用です

## 6. ゾーンごとの補助操作

### Main Deck

`Actions` ボタンから次を使えます。

- `Search`
- `Shuffle`
- `Look Top (N)`

### Cemetery / Banish / Evolve Deck

- `Search` で中身を確認できます

### Search モーダル

`Search` を押すと、そのゾーンのカード一覧が開きます。  
状況に応じて次の操作ができます。

- `Play to Field`
- `Add to Hand`
- `Reveal & Add to Hand`
- `Add to EX Area`
- `Set USED / Set UNUSED`  
  エボルヴデッキ内のカードに対して使用済み状態を切り替えます

補足:

- `Reveal & Add to Hand` は、相手に一時的に公開してから手札に加える操作です
- エボルヴカードには `Add to Hand` や `Add to EX Area` は出ません

### Look Top (N)

`Look Top (N)` では、見たカードを次の行き先に振り分けられます。

- `場 (Field)`
- `手札 (Hand)`
- `公開して手札`
- `EXエリア`
- `山上 (Top)`
- `山下 (Bottom)`
- `墓地 (Grave)`

`公開して手札` を使うと、相手に一時表示したうえで手札に入ります。

## 7. EX Area と Evolve Deck

### EX Area

- 一時的にカードを置く公開領域です
- スタッツ表示対象です
- `+C / -C` の共通カウンターも置けます

### Evolve Deck

- エボルヴカード専用のデッキです
- `Search` モーダル内で `Set USED / Set UNUSED` を切り替えられます
- 使用済みカードは表向き、未使用カードは裏向きで表示されます

## 8. カウンターとスタッツ表示

### スタッツ表示

場と EX エリアのカードには、右上に現在スタッツを表示します。

- 元の攻撃力/体力
- `+A / -A`
- `+H / -H`

を合算した値が表示されます。

### 共通カウンター

場と EX エリアのカードには、`+C / -C` で共通カウンターを置けます。  
カード右上に `Counter N` として表示されます。

### スタック時の表示

- エボルヴカードなどを上に重ねた場合、上側カードに下側カードのスタッツ変動が引き継がれて表示されます
- 下側カードは重なっている間、現在スタッツ表示を出しません

### 移動時の保持ルール

スタッツ変動と共通カウンターは、同じルールで扱われます。

- `Field -> Field` は保持
- `EX -> EX` は保持
- `EX -> Field` は保持
- `Field -> EX` はリセット
- `Field -> Cemetery / Banish / Deck / Hand` はリセット
- `EX -> Cemetery / Banish / Deck / Hand` はリセット

## 9. ターン進行

対戦中は上部ヘッダーに次が表示されます。

- 現在のターンプレイヤー
- `TURN` 数
- `Start / Main / End` フェイズ選択

主な操作:

- `End Turn`
- フェイズ変更
- `Undo Last End Turn`  
  直前の End Turn を戻すための補助です

## 10. 共有補助機能

対戦中は次の共有演出があります。

- `Toss Coin`
- `Roll Dice`
- 先攻後攻決定メッセージ
- `Reveal & Add to Hand`
- `公開して手札`

これらは相手側にも同じ結果や公開情報が表示されます。

## 11. リーダー/プレイヤーステータス

各プレイヤーのサイドパネルでは次を操作できます。

- `HP`
- `PP`
- `Max PP`
- `EP`
- `SEP`
- `Combo`

また、対戦中の補助操作として次があります。

- `Draw`
- `Mill`
- `Spawn Token`
- `Reset Game`

## 12. Reset / Undo

### Reset Game

- ゲーム全体を初期化します
- カードは元のデッキへ戻ります
- 準備状態も初期化されます

### Undo Last End Turn

- 直前の End Turn を戻すための補助機能です
- 状況によっては表示されないことがあります

## 13. 注意点

- このアプリは盤面サンドボックスです
- 公式ルールやカード効果の完全自動処理は行いません
- 攻撃制限、誘発、解決順、効果選択などは手動で進める前提です
- 非公開情報の扱いには注意してください

## 14. よく使う操作まとめ

- 手札から出す: カードにカーソルを合わせて `Play to Field`
- タップ/スタンド: 右クリック、または `REST / STAND`
- 墓地へ送る: `Cemetery`
- 消滅させる: `Banish`
- 山札に戻す: `↓Bot`
- デッキを見る: 各ゾーンの `Search`
- 山札上を見る: `Main Deck > Actions > Look Top (N)`
- 共通カウンターを置く: 場/EX のカードで `+C / -C`
