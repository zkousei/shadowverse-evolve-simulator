# P2P Manual Sync Checklist

2ブラウザでの手動確認用チェックリストです。

- `A` = Host
- `B` = Guest
- 基本確認方針
- 各操作の直後だけでなく、2-5秒待って巻き戻りがないかも確認する
- 両画面で同じ公開情報になっているか確認する
- 非公開情報が相手側に漏れていないか確認する

## 1. 接続と準備

- [ ] Aが部屋作成、Bが参加できる
- [ ] 接続後、A/Bの両画面で接続完了表示が出る
- [ ] Aで`Ready`、Bで`Ready`をほぼ同時に押しても最終状態が一致する
- [ ] Aで`Me 1st`を押したとき、両画面で先攻/後攻とEPが一致する
- [ ] Aで`Opp 1st`を押したとき、両画面で先攻/後攻とEPが一致する
- [ ] Aでランダム先攻決定を押したとき、両画面で結果が一致する
- [ ] A/Bが初手4枚ドローできる
- [ ] 自分の手札だけ自分に見え、相手には見えない
- [ ] マリガン後、両画面で手札枚数と山札枚数が一致する
- [ ] 対戦前は`Import Deck (.json)`が利用できる
- [ ] 対戦中は`Import Deck (.json)`が利用できない

## 2. ゲーム進行

- [ ] Aで`Start Game`を押したとき、両画面でゲーム開始状態が一致する
- [ ] Aで`End Turn`したとき、両画面でターンプレイヤー/ターン数/Phaseが一致する
- [ ] ターン交代時のPP回復と自動ドローが両画面で一致する
- [ ] `Undo Last End Turn`で両画面が同じ状態に戻る
- [ ] `Reset Game`で両画面が初期状態に戻る

## 3. 同時操作

- [ ] A/Bで別々のカードを同時にドラッグしても、片方の変更が消えない
- [ ] A/Bで別々のカードのカウンターを同時に変更しても、片方の変更が消えない
- [ ] Aが`End Turn`、Bが同時に別操作しても、最終状態が両画面で一致する
- [ ] 数秒待っても状態が巻き戻らない

## 4. 通常カード移動

- [ ] メインカードを場から墓地へ送れる
- [ ] メインカードを場から消滅へ送れる
- [ ] メインカードを場からデッキ下へ送れる
- [ ] 手札から場へ出せる
- [ ] 手札からEXへ出せる
- [ ] どの操作でも両画面の公開情報が一致する

## 5. エボルヴカードのルール

- [ ] エボルヴカードを場から墓地へ送ると`evolveDeck`へ戻る
- [ ] エボルヴカードを場から消滅へ送ると`evolveDeck`へ戻る
- [ ] エボルヴカードを場からデッキ系へ送ると`evolveDeck`へ戻る
- [ ] エボルヴカードを`mainDeck`へ混入できない
- [ ] メインカードを`evolveDeck`へ混入できない
- [ ] エボルヴカードが戻るとき、表向き使用済み状態のままである

## 6. スタックとトークン

- [ ] 2枚スタックを作って一緒に移動できる
- [ ] 3枚スタックを作って一緒に移動できる
- [ ] 3枚スタックを墓地へ送ったとき、墓地では重なりが解除される
- [ ] 3枚スタックをデッキへ戻したとき、デッキでは重なりが解除される
- [ ] スタック移動後にカードが孤立したり消えたりしない
- [ ] `Spawn Token`でトークン生成が両画面に反映される
- [ ] トークンを墓地/消滅/手札/デッキへ送ろうとすると完全に消える

## 7. タップ・裏返し・カウンター

- [ ] 単体カードの`Tap/Stand`が両画面で一致する
- [ ] スタックカードの`Tap/Stand`がスタック全体に反映される
- [ ] `Flip`が両画面で一致する
- [ ] ATK/HPカウンター変更が両画面で一致する
- [ ] 手札上のカードにカウンター変更が入らない

## 8. 山札操作

- [ ] `Draw Card`が両画面で一致する
- [ ] `Mill Card`が両画面で一致する
- [ ] `Shuffle`後も山札枚数が一致する
- [ ] `Look Top (N)`の結果反映が両画面で一致する

## 9. 検索モーダル

- [ ] メインデッキ検索から`Add to Hand`できる
- [ ] メインデッキ検索から`Play to Field`できる
- [ ] メインデッキ検索から`Add to EX Area`できる
- [ ] エボルヴデッキ検索で`Add to Hand`ボタンが出ない
- [ ] エボルヴデッキ検索で`Add to EX Area`ボタンが出ない
- [ ] エボルヴデッキ検索から`Play to Field`はできる
- [ ] 墓地/消滅/進化デッキ検索経由でもカード種別ルールが壊れない

## 10. HUDと補助情報

- [ ] HP変更が両画面で一致する
- [ ] PP/Max PP変更が両画面で一致する
- [ ] EP/SEP変更が両画面で一致する
- [ ] Combo変更が両画面で一致する
- [ ] Coin表示は演出として破綻しない
- [ ] Dice表示は演出として破綻しない

## 11. バグ観点メモ

- [ ] 片方だけ表示が変わっていない操作がない
- [ ] 数秒後に巻き戻る操作がない
- [ ] 相手の非公開情報が見えない
- [ ] エボルヴ/メインの分離ルールがどの導線でも守られている
- [ ] 例外操作後も次の通常操作が壊れない
